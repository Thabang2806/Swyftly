# Swyftly High-Fidelity UI Mockups

This mockup pack contains desktop and mobile screen concepts for the Swyftly transactional fashion, jewellery, accessories and beauty marketplace.

## Included screens

- Public home / discovery
- Search and category results
- Product detail
- Checkout
- Buyer AI Style Finder
- Seller dashboard
- AI Fashion Product Listing Assistant
- Seller advertising campaigns
- Admin moderation queue
- Admin finance and payouts

Each screen is available as both an HTML file and a PNG render. Open `index.html` to view the gallery.

- Deep Red: `#B42318`

## Notes

These are static mockups intended for design direction and Codex implementation prompts. They are not wired to a backend and use stylised placeholder product imagery.
